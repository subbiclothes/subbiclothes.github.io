Chat CommandsComandos de Chat

Power-user shortcuts and text commands — useful for gestures or fast execution without opening menus. Every command has a menu equivalent; nothing here is required to use the HUD. Atajos y comandos de texto para usuarios avanzados — útiles para gestos o ejecución rápida sin abrir menús. Todo comando tiene un equivalente en el menú; nada de esto es obligatorio para usar el HUD.

Notes on Specific CommandsNotas sobre Comandos Específicos

clothes.lock / .hardlock / .unlock — currently a WIP feature, roleplay/submission-themed, not a technical lock against third parties. clothes.lock [part]: anyone who isn't the owner/master sees the option greyed out as [LOCKED] and it does nothing if touched — the owner can still interact with that part normally. clothes.hardlock [part]: same lock effect, but also applied to the owner themself — though the owner can disable it any time via a direct button, so it isn't a real restriction, just a roleplay affordance. clothes.lock / .hardlock / .unlock — actualmente una función WIP, con temática de roleplay/sumisión, no es un bloqueo técnico contra terceros. clothes.lock [part]: cualquiera que no sea el dueño/master ve la opción en gris como [LOCKED] y no hace nada si se toca — el dueño puede seguir interactuando con esa parte con normalidad. clothes.hardlock [part]: mismo efecto de bloqueo, pero también aplicado al propio dueño — aunque el dueño puede desactivarlo en cualquier momento con un botón directo, así que no es una restricción real, solo un recurso de roleplay.

scfind [-v] — WIP, in active development for the DEV-4.1 line. Broadcasts a message to the whole region/parcel; other Subbi Clothes HUDs with the option enabled respond with <username> - {gender} - {bio} - {custom message} (gender, bio, and message are optional; bio only shows if set). Intended for finding people to interact/roleplay with in the region. scfind [-v] — WIP, en desarrollo activo para la línea DEV-4.1. Difunde un mensaje a toda la región/parcela; otros HUDs de Subbi Clothes con la opción activada responden con <username> - {gender} - {bio} - {custom message} (género, bio y mensaje son opcionales; la bio solo se muestra si está configurada). Pensado para encontrar gente con quien interactuar/rolear en la región.
